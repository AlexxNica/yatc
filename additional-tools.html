So, you want to edit YATC's or Tibia's picture file? Here are some of our tools for that.



Command line
These are available only through SVN and by manual compiling.
  • PicTool - GPLed command line tool to extract and replace individual images in PIC files
  • SprTool - unfinished GPLed command line tool to extract and replace images in SPR files

GUI

  • wxPicTool - GPLed GUI for PicTool - download available below
