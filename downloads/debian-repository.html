To make use of automated updates, create file /etc/apt/sources.list.d/ivucica-repository.list with following contents:
deb http://ivan.vucica.net/debrepository/ binary/
deb-src http://ivan.vucica.net/debrepository/ source/


The repository itself is not signed. Packages are however signed using GPG. Here's the public key signed by Ivan Vucica - put this in a file named "ivucica.key" and do a "gpg --import ivucica.key".
