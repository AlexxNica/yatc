Welcome to wxPicTool. This converter should be compatible with Tibia.pic files from at least version 7.1 to at least version 8.4.

This converter requires a working Tibia.pic. It cannot generate one from scratch.

This converter contains pieces of pictool: ANSI C converter for Tibia's PIC files. Readme for Pictool is included.
